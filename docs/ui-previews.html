中

tokens.css 和 style.css 使用完全不同的变量命名体系，需要在 tokens.css 中添加别名桥接

src/styles/tokens.css — 末尾追加 :root { --bg: var(--color-bg); --bg-secondary: var(--color-panel-2); --bg-tertiary: var(--color-panel); --border: var(--color-border); --text: var(--color-text); --text-secondary: var(--color-text-subtle); --accent: var(--color-brand-500); --accent-hover: var(--color-brand-600); }

这样 LegacyFrameView 等使用 --color-* 前缀的 Vue 组件，也能正确读取到 style.css 定义的 --bg、--accent 等变量值。
